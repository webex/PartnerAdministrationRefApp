# UserItem


## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**userId** | **String** | User ID to be moved. |  |
|**extension** | **String** | Extension to be moved. Only one new extension can be moved to the target location for a user. An empty value will remove the configured extension. If not provided, the existing extension will be retained. |  [optional] |
|**phoneNumber** | **String** | Phone number to be moved. Only one new phone number belonging to the target location can be assigned to a user. The phone number must follow the E.164 format. An empty value will remove the configured phone number. If not provided, the existing phone number will be moved to the target location. |  [optional] |
